# CLAUDE.md — DeepCurriculum

## Project Overview

DeepCurriculum is a Codecademy-meets-university interactive course platform. Courses are structured as full 10-week university quarter courses with lectures, problem sets, concept checks, quizzes, a midterm, and a final. All course content is **pre-generated at development time** — there are no runtime LLM API calls.

**MVP**: One course — **CS 291: Production AI Engineering** (10 weeks, 20 lectures, 9 problem sets, weekly quizzes, midterm, final, capstone project).

### Core Philosophy
- **University depth**: Graduate-level CS at a top-10 university. Dense, rigorous, with math where it belongs.
- **Pre-generated, not generative**: All content exists as files in the repo. Zero LLM calls at runtime.
- **Learn by building**: Every week has a problem set with real coding exercises that run in the browser via Pyodide.
- **No filler**: Dense or long when needed, concise when not.

## Tech Stack

| Layer | Technology | Why |
|-------|-----------|-----|
| Framework | Next.js 15 (App Router) | SSR, RSC, file-based routing |
| Database | Neon (Serverless Postgres) | Serverless, scales to zero |
| ORM | Drizzle | Type-safe, lightweight |
| Auth | Auth.js v5 + Drizzle Adapter | Credentials provider, no vendor lock-in |
| Code Editor | Monaco Editor (@monaco-editor/react) | VS Code in browser |
| Python Execution | Pyodide (WASM) | Run Python in browser, no backend |
| Styling | Tailwind CSS + shadcn/ui | Fast, accessible |
| Content | MDX + JSON files in src/content/ | Static, versioned with code |
| Math | KaTeX (rehype-katex) | Fast LaTeX |
| Syntax | rehype-highlight | Code highlighting |
| Deploy | Vercel | Zero-config |

## Architecture

```
ALL CONTENT IS STATIC (generated by Claude Code subagents at dev time):
  src/content/courses/cs291-ai-engineering/
    ├── course.json
    ├── week-01/ through week-10/
    │   ├── lecture-01.mdx     # 2000-5000 word lecture
    │   ├── lecture-02.mdx
    │   ├── concept-checks-01.json
    │   ├── concept-checks-02.json
    │   ├── problem-set.json   # 3-5 exercises with starter/solution/test code
    │   ├── quiz.json          # 8-12 questions with explanations
    │   └── readings.json
    └── solutions/

RUNTIME (zero LLM calls):
  Load MDX/JSON → Render with KaTeX + syntax highlighting
  → Monaco editor → Pyodide runs Python in browser
  → Tests validate locally → Progress saved to Neon
```

## Database Schema

```typescript
// src/lib/db/schema.ts
import { pgTable, uuid, text, timestamp, integer, jsonb, boolean } from 'drizzle-orm/pg-core';

export const userProfiles = pgTable('user_profiles', {
  id: uuid('id').primaryKey().defaultRandom(),
  userId: text('user_id').unique().notNull(),
  displayName: text('display_name'),
  email: text('email'),
  passwordHash: text('password_hash'),
  totalXp: integer('total_xp').default(0),
  streakDays: integer('streak_days').default(0),
  lastActiveAt: timestamp('last_active_at'),
  createdAt: timestamp('created_at').defaultNow(),
});

export const courses = pgTable('courses', {
  id: uuid('id').primaryKey().defaultRandom(),
  slug: text('slug').unique().notNull(),
  title: text('title').notNull(),
  code: text('code').notNull(),
  description: text('description').notNull(),
  quarterLength: integer('quarter_length').default(10),
  createdAt: timestamp('created_at').defaultNow(),
});

export const weeks = pgTable('weeks', {
  id: uuid('id').primaryKey().defaultRandom(),
  courseId: uuid('course_id').references(() => courses.id).notNull(),
  slug: text('slug').notNull(),
  weekNumber: integer('week_number').notNull(),
  title: text('title').notNull(),
  description: text('description').notNull(),
});

export const lessons = pgTable('lessons', {
  id: uuid('id').primaryKey().defaultRandom(),
  weekId: uuid('week_id').references(() => weeks.id).notNull(),
  slug: text('slug').notNull(),
  title: text('title').notNull(),
  type: text('type').notNull(),
  orderIndex: integer('order_index').notNull(),
  contentPath: text('content_path').notNull(),
  estimatedMinutes: integer('estimated_minutes').default(30),
});

export const lessonProgress = pgTable('lesson_progress', {
  id: uuid('id').primaryKey().defaultRandom(),
  userId: text('user_id').notNull(),
  lessonId: uuid('lesson_id').references(() => lessons.id).notNull(),
  completed: boolean('completed').default(false),
  score: integer('score'),
  startedAt: timestamp('started_at'),
  completedAt: timestamp('completed_at'),
  timeSpentSeconds: integer('time_spent_seconds').default(0),
});

export const exerciseSubmissions = pgTable('exercise_submissions', {
  id: uuid('id').primaryKey().defaultRandom(),
  userId: text('user_id').notNull(),
  lessonId: uuid('lesson_id').references(() => lessons.id).notNull(),
  exerciseIndex: integer('exercise_index').notNull(),
  code: text('code').notNull(),
  passed: boolean('passed').default(false),
  testResults: jsonb('test_results'),
  submittedAt: timestamp('submitted_at').defaultNow(),
});
```

## Content File Formats

### Lecture (MDX)
```mdx
---
title: "Lecture Title"
week: 1
lecture: 1
estimatedMinutes: 50
---

Full lecture content with $LaTeX$ math, ```python code blocks```,
and <ConceptCheck id="cc-1" /> markers for inline checks.
```

### Concept Checks (JSON)
```json
{
  "checks": [
    {
      "id": "cc-1",
      "type": "multiple_choice",
      "question": "...",
      "options": ["A", "B", "C", "D"],
      "correct": 1,
      "explanation": "..."
    }
  ]
}
```

### Problem Set (JSON)
```json
{
  "title": "Problem Set N",
  "exercises": [
    {
      "id": "ps1-ex1",
      "title": "...",
      "points": 25,
      "difficulty": "intermediate",
      "description": "Markdown instructions...",
      "instructions": "Detailed instructions with math...",
      "starterCode": "# Complete Python code with imports...",
      "solutionCode": "# Full solution...",
      "testCode": "# 4-8 assertions...\nprint('All tests passed!')",
      "hints": ["hint1", "hint2", "hint3"]
    }
  ]
}
```

### Quiz (JSON)
```json
{
  "title": "Quiz N",
  "timeLimit": 20,
  "passingScore": 70,
  "questions": [
    {
      "id": "q1",
      "type": "multiple_choice",
      "points": 10,
      "question": "...",
      "options": ["A", "B", "C", "D"],
      "correct": 2,
      "explanation": "Detailed teaching explanation..."
    }
  ]
}
```

## Content Quality Standards

- **Lectures**: 2000-5000 words, first principles with real math (LaTeX), runnable Python, 2-4 concept checks, cite papers by name.
- **Exercises**: Starter + solution + 4-8 test assertions + 2-3 hints. All code runs in Pyodide (numpy, pandas, math, collections, json, dataclasses).
- **Quizzes**: 8-12 questions with thorough explanations. Midterm 18q/45min. Final 22q/60min.
- **Tone**: Brilliant professor. Rigorous, direct, occasionally witty, never condescending.
